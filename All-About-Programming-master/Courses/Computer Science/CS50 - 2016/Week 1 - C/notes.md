# C

When clicked say hello world in c

```c
#include <stdio.h>

int main(void)
{
  printf("hello, world\n");
}
```

Loop

```c
while (true)
{
  printf("hello, world\n");
}
```

Repeat 50 say hello world

```c
for (int i = 0; i < 50; i++)
{
  printf("hello, world\n");
}
```
